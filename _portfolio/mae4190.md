## Course Overview

MAE 4190: Fast Robots is a hands-on course at Cornell University focusing on designing fast autonomous cars and implementing dynamic behaviors through practical experience with sensors, actuators, and control systems.

**Course Period**: Spring 2024

## Labs Completed

This project includes 12 comprehensive labs covering:

1. **Artemis and Bluetooth** - Microcontroller setup and wireless communication
2. **IMU** - Inertial Measurement Unit integration and sensor fusion
3. **ToF** - Time-of-Flight distance sensing
4. **Motor Drivers and Open Loop Control** - Motor control fundamentals
5. **Linear PID and Linear Interpolation** - Closed-loop control implementation
6. **Orientation PID** - Angular position control
7. **Kalman Filtering** - State estimation and sensor fusion
8. **Stunts!** - Dynamic maneuvers and performance optimization
9. **Mapping** - Environment mapping using sensors
10. **Localization (sim)** - Position estimation in simulation
11. **Localization (real)** - Real-world localization implementation
12. **Planning and Execution** - Path planning and autonomous navigation
